Convert a pdf to multiple photos:

$ convert -scale 2048x1447 "your_book.pdf" page%d.jpg --> SLOW

Faster:  GhostScript:
gs -sDEVICE=png16m -o book/page-%d.png -r600x600 book.pdf
THEN:

convert -resize 1024x1024 -quality 95% book/* book/page%d.jpg

CONVERT IMAGES

convert -limit memory 512 -resize 1024x1024 -quality 95% book/* book/page%d.jpg



env MAGICK_TMPDIR=/media/Windows/var convert -limit memory 512MiB -limit map 1000MiB -resize 507x507 -quality 90% ./*.jpg resized/photo%02d.jpg

MAKING SAME-SIZE THUMBNAILS


convert -limit memory 512MiB -limit map 1000MiB -thumbnail "50x50^>" -gravity center -crop 50x50+0+0 +repage -quality 80% ./*.jpg thumbs/thumbs%02d.jpg

use %02d instead of %d to insert leading zeros (fixed 2 digits)

Ref: http://www.imagemagick.org/Usage/files/



Start a http server at any folder


python -m SimpleHTTPServer [port]
