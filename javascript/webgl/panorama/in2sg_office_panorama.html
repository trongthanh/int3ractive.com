In2 Office Walkthrough with Equirectangular Panorama - An experiment by Thanh Tran
The example is based on a three.js example.

Usage: Drag the screen to navigate around. Click on named markers for more person info. Click on observer marker to change position.
List of employees is updated until 11/11/2011

Note. You need a modern browser that supports Canvas and latest Javascript specifications for this to run the way it is intended.
For example, Google Chrome 14+ or Firefox 7+.

This demo will render with WebGL by default. If WebGL is not available for the browser, it will fallback to 2D Canvas rendering which is slower.
Current renderer: NO CANVAS
